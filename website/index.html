Wings-core.

Experimental self-hosted state sync & version system.

Core Functionality

Wings-core allows you to:

  • Initialize a project with a unique identifier
  • Push and pull project versions
  • Even from your phone! (Coming soon)
  • Track and manage project state changes
  • It is silly :)

Interested? Download here!

Download!

Available Commands!

  • Wings-core init: Initializes a new project with a unique identifier.
  • Wings-core push: Pushes the current project state to the remote repository.
  • Wings-core pull: Pulls the latest project state from the remote repository.
  • Wings-core status: Shows the current status of the project and any changes.
