Wings-core.

Experimental self-hosted state sync & version system wings-core is the foundation of a personal, self-reliant alternative to hosted code platforms. Instead of relying on GitHub, this project introduces its own CLI, server, and protocol for pushing and pulling project state — fully under your control.

Core Functionality

Wings-core allows you to:

  • Initialize a project with a unique identifier
  • Push and pull project versions to/from a self-hosted server
  • Even from your phone! (Coming soon)
  • Track and manage project state changes (See commands below.)
